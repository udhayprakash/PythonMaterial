Imagine in your code you have the following:

yield{
    'Article Title':title,
    'Date':date,
    'Author':author,
    'Text':text
    }


However, in your output CSV, you have this order:

Date | Text | Article Title | Author


Question:

What to do to get the exported items in the preferred order?


Answer:

Open the file settings.py found in the Scrapy project and add this line:

FEED_EXPORT_FIELDS = ['Article Title', 'Date', 'Author', 'Text']

This will give you the order you need.


Try this in your code, and if you have questions, please feel free to send them to the Q&A section of this course.
