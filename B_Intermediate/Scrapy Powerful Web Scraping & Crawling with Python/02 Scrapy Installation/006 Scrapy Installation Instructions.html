Linux (Ubuntu):

1. Install dependencies as follows:

sudo apt-get install python-dev python-pip libxml2-dev libxslt1-dev zlib1g-dev libffi-dev libssl-dev

2. Open Terminal and install Scrapy using pip as follows:

sudo pip install scrapy


Mac:

1. Open Terminal and install Scrapy using pip as follows:

sudo pip install scrapy


Windows:

1. Download and install Anaconda for Windows

2. Open CMD and run the following command:

pip install scrapy

If this does not work, try:

conda install -c conda-forge scrapy



Installing IPython:


In Terminal or CMD, install IPython using pip as follows:

pip install ipython

Start with sudo on Linux and Mac.


That is it! If you have any issues while installing or using Scrapy, please send them to the Q&A section of this course.
