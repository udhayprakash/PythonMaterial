Here are some easy methods to get XPath.


1- Google Chrome

  1. Open the web page in Google Chrome.
  2. Select the text portion you want to extract.
  3. Right-click, and select "Inspect".
  4. Select the HTML code you need, and select "Copy" and then "Copy XPath".
  5. Paste the XPath to your code, test, and edit it, if necessary.
  6. Note that this method copy the "id" but you can change it to the "class" of the same portion if that will work better.




2- Firefox

Works in a way that is very similar to Google Chrome.



3- XPath Helper (Chrome Extension)

  1. Install XPath Helper Chrome extension (link). Restart Chrome.
  2. Open a new tab and navigate to any webpage.
  3. To open the XPath Helper console, click the XPath Helper button in the toolbar or press Ctrl-Shift-X (or Command-Shift-X on OS X).
  4. Move your mouse over the portion you want to extract, and hold down Shift. The query box will show the XPath query for the element below the mouse pointer.
  5. Note that it extracts the full XPath which you can use as is, or you can delete the first part and replace it with an extra slash.
  6. You can edit the XPath query directly in the console. The results box will immediately reflect your changes.


4- XPath Tester

It is to write your XPath strings and try it.

  1. Open XPath Tester (link).
  2. Copy paste the HTML source code into the first field "XML Input".
  3. Type your XPath in the field "XPath expression".
  4. Click the button "TEST XPATH".
  5. Check the result.
